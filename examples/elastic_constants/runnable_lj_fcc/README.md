# Runnable Starter: FCC Elastic Constants

This folder contains a small, original LAMMPS example for estimating elastic
constants from finite strain. It uses a Lennard-Jones FCC crystal so students
can run it without downloading a separate EAM potential file.

This is a teaching example. It is meant to show the workflow and syntax, not to
replace the official LAMMPS `ELASTIC` example or a careful production elastic
constant calculation.

## Files

- `in.relax_lj`: builds and relaxes an FCC Lennard-Jones crystal.
- `in.c11_c12_lj`: applies a small x strain and estimates `C11` and `C12`.
- `in.c44_lj`: applies a small shear strain and estimates `C44`.

## Run

From this folder:

```bash
lmp -in in.relax_lj
lmp -in in.c11_c12_lj
lmp -in in.c44_lj
```

If your executable has another name:

```bash
lammps_serial -in in.relax_lj
```

Run `in.relax_lj` first. The other two scripts read
`relaxed_lj_fcc.data`, which is generated by the relaxation step.

## What To Look For

In the terminal or `log.lammps`, look for lines like:

```text
C11 estimate
C12 estimate
C44 estimate
```

The values are in Lennard-Jones reduced pressure units because this starter
uses `units lj`. For real metals in `units metal`, pressure is usually in bar,
and elastic constants are often converted to GPa.

## How The Method Works

For a small strain:

```text
elastic constant ~= stress change / strain
```

LAMMPS reports pressure tensor components, and pressure has the opposite sign
from the usual tensile stress convention. That is why the scripts use a minus
sign when calculating `C11`, `C12`, and `C44`.

## How To Adapt This To Ni

Replace the Lennard-Jones potential section:

```lammps
pair_style      lj/cut 2.5
pair_coeff      1 1 1.0 1.0 2.5
```

with an EAM potential section:

```lammps
pair_style      eam/alloy
pair_coeff      * * /path/to/Ni_potential.eam.alloy Ni
```

Also change:

```lammps
units           lj
lattice         fcc 1.55
```

to something appropriate for Ni:

```lammps
units           metal
lattice         fcc 3.52
```

After switching to `units metal`, remember that pressure is in bar unless the
script converts it. Use:

```text
1 GPa = 10000 bar
```
